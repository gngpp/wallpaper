### 壁纸下载器

#### 简介
> wallpaper 是一个壁纸爬取下载程序。本项目仅需要Java11即可运行

> 1.clone 2.修改配置文件 3.编译 4.运行

#### 支持壁纸网站（目前支持）
- [彼岸壁纸](https://pic.netbian.com)
    > 通过模拟登录获取Cookie请求获取HTML静态文档，实现分页解析DOM和CSS选择器，主要功能有：按类分页下载、自动分页下载。
    > 网站每天最大下载量（200张4K/5K/8K）
